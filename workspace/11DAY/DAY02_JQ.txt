attr() 속성 값 읽기

    HTML 태그 요소의 특정 속성값을 얻기 위해서 attr() 사용
        $(요소).attr('속성이름')

attr() 속성 값 변경/추가

    $(요소).attr( '속성이름' , '값' ) 

다중처리 

    $(요소).attr({
        속성이름:값
        속성이름:값
        속성이름:값
    });

CSS 속성제어

    속성값 읽어오기
        $(요소).css(속성이름)
    속성값 적용
        $(요소).css(속성이름, 값)
    CSS 클래스의 적용여부
        해당 요소에 class가 적용되어있는지 boolean 값으로 확인하기 

        $(요소).hasClass('클래스명')
            클래스가 적용되면 T
            클래스가 적용안되면 F

    클래스 추가적용

        2 개 이상의 클래스를 한번에 적용할 경우  공백으로 구분하여 지정 
          $(요소).addClass('클래스이름')

    클래스 제거

        2 개 이상의 클래스에 대한 처리일 경우 공백으로 구분
            $(요소).removeClass('클래스이름)

    $(요소).toggleClass('클래스 이름')
        있는것은 제거, 비어있으면 추가
